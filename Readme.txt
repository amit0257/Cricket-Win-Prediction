BCCI has hired external analytics consulting firm for data analytics.The major objective of this tie-up is to extract actionable insights from the historical match data and make strategic changes to make India win. The primary objective is to create Machine Learning models which correctly predict a win for the Indian Cricket Team. Once a model is developed then you have to extract actionable insights and recommendations.The purpose of this whole exercise is to explore the dataset and to understand how team India performs under different conditions and make strategic changes to make India Win. The data consists of 2930 games played by team India with 9 unique opponents. Analyzing different attributes of the game can help us understand what are the important attributes which can lead to India's Win
